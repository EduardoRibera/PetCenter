## Requisitos

Solo necesitas un navegador web moderno. No requiere bases de datos locales complejas instaladas en tu equipo, ya que la integración con la base de datos se maneja a través de la API del cliente web de Supabase.

## Ejecución Local

1. Descarga o clona el repositorio desde tu cuenta de GitHub.
2. Abre la carpeta `Proyecto/` en tu editor de código preferido (ej. VS Code).
3. Abre el archivo `index.html` en tu navegador.
4. Para una experiencia óptima de desarrollo (evitando bloqueos de CORS o rutas), utiliza una extensión de servidor local como **Live Server** en VS Code.
