# PetCenter Petshop

PetCenter es una plataforma integral para el cuidado de mascotas. El sitio combina una tienda en línea (Pet Shop) con un completo sistema de reservas para servicios como Hotel Pet, Veterinaria, Peluquería y Guardería.

## Características Clave

- **Tienda en línea completa**: Carrito de compras persistente a través del almacenamiento local.
- **Servicios**: Reserva de turnos con cotización dinámica y envío de datos en tiempo real.
- **Privacidad y Seguridad**: Integración con APIs mediante almacenamiento local de llaves (`localStorage`).
- **Automatización**: Procesamiento de pedidos e interacciones a través de flujos de **n8n**.
- **Base de Datos**: Sincronización en tiempo real con **Supabase** para el inventario de productos.
