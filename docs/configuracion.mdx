## Panel de Configuración

El proyecto utiliza la página `config.html` para enlazar de manera segura el frontend con tus servicios en la nube. 

### 1. Configurar Supabase

Para habilitar la tienda con productos en tiempo real:
1. Ve a [Supabase.com](https://supabase.com) y crea un proyecto.
2. Crea una tabla llamada `productos` con los siguientes campos principales:
   - `id` (Texto/UUID, Primary Key)
   - `nombre` (Texto)
   - `descripcion` (Texto)
   - `precio` (Numérico)
   - `imagen_url` (Texto)
   - `categoria` (Texto)
   - `animal` (Texto)
   - `marca` (Texto)
   - `stock` (Entero)
3. Copia la **URL de Supabase** y la **Anon Key** (Clave Pública).
4. Pégalas en el formulario en la pestaña de configuración del sitio (`config.html`).
5. Presiona **Probar Supabase** para verificar la conexión.

### 2. Configurar Webhooks de n8n

Para la automatización de pedidos, reservas y mensajes de contacto:
1. Diseña un flujo (workflow) en tu instancia de n8n que empiece con un nodo **Webhook**.
2. Copia la URL del Webhook generada por n8n.
3. Pégala en el campo correspondiente en `config.html`:
   - **Webhook Pedidos**: Para las compras de la tienda.
   - **Webhook Reservas**: Para el agendamiento de citas de servicios.
   - **Webhook Contacto**: Para mensajes de ayuda u otros.
4. Haz clic en **Guardar Credenciales**.
