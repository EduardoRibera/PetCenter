## Funcionamiento de la Tienda

La sección de la tienda (`shop.html`) muestra los productos de manera dinámica:
- Si **Supabase** está configurado correctamente, cargará el catálogo de productos de tu base de datos.
- Si no hay credenciales guardadas, la tienda utilizará una lista de productos de respaldo (**fallback**) para que el sitio siempre sea funcional y visualizable.

### Filtros
Los usuarios pueden filtrar los productos por:
- Categoría (Alimentos, Farmacia, Accesorios, Juguetes)
- Búsqueda por palabras clave.

## El Carrito de Compras

El carrito (`carrito.html`) permite:
1. Ver los artículos agregados.
2. Incrementar o decrementar cantidades.
3. Calcular subtotales e impuestos automáticamente.
4. Realizar la orden (Checkout).

Al finalizar la orden, se enviará un JSON estructurado con el detalle del pedido al **Webhook de Pedidos de n8n** configurado para procesar la venta (ej. enviar confirmaciones por correo, registrar en planillas, etc.).
